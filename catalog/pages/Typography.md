Für Überschriften wird die Schriftart **Broshk** verwendet. Für den Fließtext wird die Schriftart **Century Gothic** verwendet.

```type|single
{

  "font": "Broshk",
  "headings": [72]
}
```

```type
{
    "paragraphs": ["16/24"],
    "font": "Century Gothic"
}
```
